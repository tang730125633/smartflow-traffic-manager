## 添加自定义视频

1. 将视频文件复制到 `sftm_server/static/videos/` 目录
2. 支持的格式：MP4, WebM, OGG
3. 建议分辨率：1920x1080 或 1280x720
4. 建议时长：10-30秒的循环视频

## 视频切换功能

在仪表板页面，您可以：
- 点击不同的按钮切换视频源
- 使用"本地视频"按钮播放本地文件
- 使用"演示视频"按钮查看模拟数据

## 故障排除

如果视频无法播放：
1. 检查网络连接
2. 尝试切换其他视频源
3. 使用演示模式查看功能
4. 检查浏览器是否支持视频播放
