📋 最简流程（3 步走）
═══════════════════════════════════════════════════════════════════════════

如果你想用最快的方式运行起来，按这个流程：


第 1 步：下载软件
─────────────────────────────────────────────────────
打开《软件下载清单.txt》，下载所有 5 个软件：
  ✓ Java 11
  ✓ Maven 3.9
  ✓ MySQL 8.0
  ✓ Redis
  ✓ Python 3.10


第 2 步：安装软件
─────────────────────────────────────────────────────
按照《【必读】零基础运行教程.txt》的【第二部分】安装每个软件

⚠️ 重要提示：
  - Java 和 Maven 需要配置环境变量
  - MySQL 密码必须设为：root
  - Redis 和 Python 安装时要勾选"Add to PATH"


第 3 步：运行项目
─────────────────────────────────────────────────────
双击运行：一键启动.bat
选择：1（最小模式）
等待启动完成（5-10 分钟）
打开浏览器访问：http://localhost:8080


═══════════════════════════════════════════════════════════════════════════
🔍 检查清单
═══════════════════════════════════════════════════════════════════════════

在启动项目之前，确保完成以下检查：

□ 已安装 Java 11，并配置环境变量 JAVA_HOME
□ 已安装 Maven 3.9，并配置环境变量 MAVEN_HOME
□ 已在 Maven settings.xml 中配置阿里云镜像
□ 已安装 MySQL 8.0，root 密码为 root
□ 已安装 Redis，并添加到 PATH
□ 已安装 Python 3.10，并添加到 PATH
□ 已配置 pip 清华镜像
□ 已运行 sql\init.sql 初始化数据库
□ 已运行 pip install -r requirements.txt 安装 Python 依赖

✅ 验证方法：双击运行 check-services.bat

❓ 常见问题
═══════════════════════════════════════════════════════════════════════════

Q1: 我的电脑配置够吗？
─────────────────────────────────────────────────────
最低配置：Windows 7+, 4GB 内存, 10GB 硬盘空间
推荐配置：Windows 10+, 8GB 内存, 20GB 硬盘空间

如果内存不足，选择"最小模式"启动（只启动核心服务）


Q2: 需要安装 Docker 吗？
─────────────────────────────────────────────────────
不需要！这个教程专门为没有 Docker 的环境设计。
所有服务都可以直接在 Windows 上运行。


Q3: 需要梯子（VPN）吗？
─────────────────────────────────────────────────────
不需要！所有下载链接都是国内镜像：
  - Java：华为云
  - Maven 依赖：阿里云
  - MySQL：清华大学镜像
  - Python 包：清华大学镜像


Q4: 第一次启动为什么这么慢？
─────────────────────────────────────────────────────
第一次启动需要下载大量依赖包（约 500MB），需要 20-40 分钟。
之后启动只需要 2-5 分钟。


Q5: 出现错误怎么办？
─────────────────────────────────────────────────────
1. 查看《【必读】零基础运行教程.txt》的【常见问题解决】章节
2. 运行 check-services.bat 检查环境
3. 记录完整的错误信息，向技术人员求助
