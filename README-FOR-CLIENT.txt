========================================================
      SmartFlow Traffic Manager - Client Installation
========================================================

GETTING STARTED (3 Simple Steps):
---------------------------------

Step 1: Extract files
  - Extract the downloaded zip file to a folder
  - Example: C:\SmartFlow\


Step 2: Double-click "START.bat"
  - This will check and install everything automatically
  - First run takes 5-10 minutes
  - Follow the on-screen instructions


Step 3: Register and Login
  - Browser will open automatically
  - Click "Register" to create your account
  - Login and start using the system


WHAT GETS INSTALLED:
-------------------
- Python 3.11 (if not already installed)
- Required libraries (Flask, etc.)
- Database (automatic setup)


SYSTEM REQUIREMENTS:
-------------------
- Windows 7, 8, 10, or 11
- 500 MB free disk space
- Internet connection (first-time setup only)
- Administrator rights (for Python installation)


IMPORTANT NOTES:
----------------
- KEEP THIS WINDOW OPEN while using the application
- First run takes 5-10 minutes (one-time setup)
- You need administrator rights to install Python
- After setup, future starts only take 10-20 seconds


IF SOMETHING GOES WRONG:
-------------------------

Problem: "Python not found"
Solution:
  1. Make sure you're connected to internet
  2. Download Python from: https://www.python.org/downloads/
  3. Install with "Add Python to PATH" checked
  4. Run START.bat again

Problem: "Dependencies failed to install"
Solution:
  1. Check your internet connection
  2. Close antivirus software temporarily
  3. Run START.bat again

Problem: "Application won't start"
Solution:
  1. Make sure you ran START.bat first (one-time setup)
  2. Wait 10-20 seconds for startup
  3. Try running START.bat again
